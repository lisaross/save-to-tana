Save to Tana - Options

Setup Instructions

  1. Navigate to your Tana node with the #save-to-tana supertag
  2. Click the extraction button that appears next to the node (info icon)
  3. The configuration will be copied to your clipboard
  4. Paste the configuration in the "Paste Configuration" tab below
  5. Enter your Tana API Token
  6. Enter your Target Node ID (required)
  7. Save your settings
Click the extraction button on your Tana node, then paste the copied configuration here.
